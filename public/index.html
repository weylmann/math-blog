什么是人工智能？

人工智能（Artificial Intelligence，简称 AI）是使计算机具有人类智能的技术学科。

人工智能的分支

1. 机器学习 (Machine Learning)

让计算机通过数据学习并改进。

2. 深度学习 (Deep Learning)

使用神经网络模拟人脑工作方式。

3. 自然语言处理 (NLP)

让计算机理解和生成人类语言。

推荐的入门路径

  1. Python 基础 - 学会编程
  2. 数学基础 - 线性代数、概率论
  3. 机器学习 - 了解基本算法
  4. 深度学习 - 神经网络
