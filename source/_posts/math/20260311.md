每日一题 2026-03-11

**题目：**  
设 $O$ 是 $\triangle ABC$ 的内心，已知 $\cos \angle BAC = \dfrac{1}{3}$，且 $\overrightarrow{AO} = x\overrightarrow{AB} + y\overrightarrow{AC}$，求 $x + y$ 的最大值。

**几何示意图：**  
考虑 $\triangle ABC$，$O$ 为内心，$AO$ 延长交 $BC$ 于 $D$，则 $AD$ 是 $\angle BAC$ 的平分线。

<details><summary> 参考解答</summary>

**解答思路：**

1. **角平分线向量表示**  
   延长 $AO$ 交 $BC$ 于 $D$，则 $AD$ 是 $\angle BAC$ 的平分线。  
   由角平分线性质：$\frac{BD}{DC} = \frac{AB}{AC} = \frac{c}{b}$  
   $\therefore \overrightarrow{AD} = \frac{b}{b+c}\overrightarrow{AB} + \frac{c}{b+c}\overrightarrow{AC}$

2. **内心位置比例**  
   内心 $O$ 在 $AD$ 上，且 $\frac{AO}{OD} = \frac{AB+AC}{BC} = \frac{b+c}{a}$  
   $\therefore \frac{AO}{AD} = \frac{b+c}{a+b+c}$，即 $\overrightarrow{AO} = \frac{b+c}{a+b+c}\overrightarrow{AD}$

3. **代入得系数**  
   $\overrightarrow{AO} = \frac{b+c}{a+b+c}\left(\frac{b}{b+c}\overrightarrow{AB} + \frac{c}{b+c}\overrightarrow{AC}\right) = \frac{b}{a+b+c}\overrightarrow{AB} + \frac{c}{a+b+c}\overrightarrow{AC}$  
   与 $\overrightarrow{AO} = x\overrightarrow{AB} + y\overrightarrow{AC}$ 比较得：  
   $x = \frac{b}{a+b+c},\quad y = \frac{c}{a+b+c}$  
   $\therefore x+y = \frac{b+c}{a+b+c}$

4. **利用余弦定理**  
   已知 $\cos A = \dfrac{1}{3}$，由余弦定理：  
   $a^2 = b^2 + c^2 - 2bc\cos A = b^2 + c^2 - \dfrac{2}{3}bc$

5. **求最大值**  
   令 $t = \frac{b}{c} > 0$，则：  
   $x+y = \frac{c(t+1)}{c\sqrt{t^2+1-\frac{2}{3}t} + c(t+1)} = \frac{t+1}{\sqrt{t^2-\frac{2}{3}t+1} + t+1}$

   当 $t=1$（即 $b=c$，等腰三角形）时，分母最小，$x+y$ 最大。

6. **计算最大值**  
   当 $b=c=1$ 时，$a = \sqrt{1^2+1^2-2\times\frac{1}{3}} = \sqrt{2-\frac{2}{3}} = \sqrt{\frac{4}{3}} = \frac{2}{\sqrt{3}}$  
   $x+y = \frac{1+1}{\frac{2}{\sqrt{3}}+1+1} = \frac{2}{2+\frac{2}{\sqrt{3}}} = \frac{1}{1+\frac{1}{\sqrt{3}}} = \frac{\sqrt{3}}{\sqrt{3}+1} = \frac{\sqrt{3}(\sqrt{3}-1)}{2} = \frac{3-\sqrt{3}}{2}$

**答案：** $\boxed{\frac{3-\sqrt{3}}{2}}$
