每日一题 2026-03-17

在 $\triangle ABC$ 中，求 $\sin^2 A + \sin^2 B + \sin^2 C$ 的最大值。

<details>
<summary>参考解答</summary>

<strong>答案：</strong> $\dfrac{9}{4}$

<strong>解析</strong>

设 $f = \sin^2 A + \sin^2 B + \sin^2 C$。

利用降幂公式 $\sin^2 x = \dfrac{1 - \cos 2x}{2}$，得：
$$f = \frac{1 - \cos 2A}{2} + \frac{1 - \cos 2B}{2} + \frac{1 - \cos 2C}{2} = \frac{3}{2} - \frac{1}{2}(\cos 2A + \cos 2B + \cos 2C)$$

在任意三角形中，有恒等式：
$$\cos 2A + \cos 2B + \cos 2C = -1 - 4\cos A \cos B \cos C$$

代入得：
$$f = \frac{3}{2} - \frac{1}{2}(-1 - 4\cos A \cos B \cos C) = 2 + 2\cos A \cos B \cos C$$

在三角形中，$\cos A \cos B \cos C$ 的最大值出现在等边三角形时（即 $A = B = C = 60^\circ$），此时：
$$\cos A = \cos B = \cos C = \frac{1}{2}$$
$$\cos A \cos B \cos C = \frac{1}{8}$$

因此：
$$f_{\text{max}} = 2 + 2 \times \frac{1}{8} = \frac{9}{4}$$

验证：当 $A = B = C = 60^\circ$ 时，
$$\sin^2 A + \sin^2 B + \sin^2 C = 3 \times \left(\frac{\sqrt{3}}{2}\right)^2 = \frac{9}{4}$$

故 $\sin^2 A + \sin^2 B + \sin^2 C$ 的最大值为 $\dfrac{9}{4}$。
