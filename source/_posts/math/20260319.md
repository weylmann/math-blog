在 $\triangle ABC$ 中，$\angle B = 60^\circ$，$AC = 1$。点 $D$ 在 $AB$ 上满足 $\vec{BD} = \dfrac{1}{3}\vec{BA}$，$E$ 在 $CD$ 上满足 $\vec{DE} = \dfrac{1}{4}\vec{DC}$，$F$ 是 $AC$ 的中点。求 $\vec{BE} \cdot \vec{BF}$ 的最大值。

<details>
<summary>参考解答</summary>

<strong>解析</strong>

设 $\vec{BA} = \vec{a}$，$\vec{BC} = \vec{b}$。

由于 $\angle B = 60^\circ$，所以 $\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos 60^\circ = \dfrac{1}{2}|\vec{a}||\vec{b}|$。

由于 $AC = 1$，且 $\vec{AC} = \vec{BC} - \vec{BA} = \vec{b} - \vec{a}$，所以：
$$|\vec{b} - \vec{a}|^2 = 1$$
$$|\vec{b}|^2 + |\vec{a}|^2 - 2\vec{a} \cdot \vec{b} = 1$$
$$|\vec{a}|^2 + |\vec{b}|^2 - |\vec{a}||\vec{b}| = 1 \quad (1)$$

接下来表示各点的向量（以点 $B$ 为原点）：

- 点 $D$：$\vec{BD} = \dfrac{1}{3}\vec{BA} = \dfrac{1}{3}\vec{a}$，所以 $\vec{D} = \dfrac{1}{3}\vec{a}$
- 点 $C$：$\vec{C} = \vec{b}$
- 向量 $\vec{DC} = \vec{C} - \vec{D} = \vec{b} - \dfrac{1}{3}\vec{a}$
- 点 $E$：$\vec{DE} = \dfrac{1}{4}\vec{DC} = \dfrac{1}{4}\left(\vec{b} - \dfrac{1}{3}\vec{a}\right) = \dfrac{1}{4}\vec{b} - \dfrac{1}{12}\vec{a}$
- 所以 $\vec{E} = \vec{D} + \vec{DE} = \dfrac{1}{3}\vec{a} + \dfrac{1}{4}\vec{b} - \dfrac{1}{12}\vec{a} = \dfrac{1}{4}\vec{a} + \dfrac{1}{4}\vec{b}$
- 点 $A$：$\vec{A} = \vec{a}$
- 点 $F$（$AC$ 中点）：$\vec{F} = \dfrac{1}{2}(\vec{A} + \vec{C}) = \dfrac{1}{2}(\vec{a} + \vec{b})$

计算 $\vec{BE}$ 和 $\vec{BF}$：
- $\vec{BE} = \vec{E} - \vec{B} = \dfrac{1}{4}\vec{a} + \dfrac{1}{4}\vec{b}$
- $\vec{BF} = \vec{F} - \vec{B} = \dfrac{1}{2}\vec{a} + \dfrac{1}{2}\vec{b}$

计算点积：
$$\vec{BE} \cdot \vec{BF} = \left(\dfrac{1}{4}\vec{a} + \dfrac{1}{4}\vec{b}\right) \cdot \left(\dfrac{1}{2}\vec{a} + \dfrac{1}{2}\vec{b}\right)$$
$$= \dfrac{1}{8}|\vec{a}|^2 + \dfrac{1}{8}\vec{a} \cdot \vec{b} + \dfrac{1}{8}\vec{b} \cdot \vec{a} + \dfrac{1}{8}|\vec{b}|^2$$
$$= \dfrac{1}{8}\left(|\vec{a}|^2 + |\vec{b}|^2 + 2\vec{a} \cdot \vec{b}\right)$$
$$= \dfrac{1}{8}\left(|\vec{a}|^2 + |\vec{b}|^2 + |\vec{a}||\vec{b}|\right) \quad (\text{因为 } \vec{a} \cdot \vec{b} = \dfrac{1}{2}|\vec{a}||\vec{b}|)$$

由约束条件 $(1)$ 得：$|\vec{a}|^2 + |\vec{b}|^2 = 1 + |\vec{a}||\vec{b}|$，代入上式：
$$\vec{BE} \cdot \vec{BF} = \dfrac{1}{8}\left(1 + |\vec{a}||\vec{b}| + |\vec{a}||\vec{b}|\right) = \dfrac{1}{8}\left(1 + 2|\vec{a}||\vec{b}|\right)$$

现在需要最大化 $|\vec{a}||\vec{b}|$。设 $x = |\vec{a}|$，$y = |\vec{b}|$，则约束条件为：
$$x^2 + y^2 - xy = 1$$

由基本不等式 $x^2 + y^2 \geq 2xy$，得：
$$2xy - xy \leq 1 \implies xy \leq 1$$

当且仅当 $x = y$ 时取等号。此时 $x^2 + x^2 - x^2 = x^2 = 1$，所以 $x = y = 1$。

因此 $|\vec{a}||\vec{b}|$ 的最大值为 $1$。

<strong>答案：</strong>
$$\max(\vec{BE} \cdot \vec{BF}) = \dfrac{1}{8}(1 + 2 \times 1) = \dfrac{3}{8}$$
