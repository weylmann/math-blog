已知三个不共线的单位向量 $\mathbf{a}, \mathbf{b}, \mathbf{c}$ 满足 $\mathbf{a} + \mathbf{b} + \mathbf{c} = \mathbf{0}$，求 $|-2\mathbf{a} + t\mathbf{b} + (1-t)\mathbf{c}|$ 在 $0 \le t \le 1$ 时的取值范围。

<details><summary>参考思路</summary>

由于 $\mathbf{a}, \mathbf{b}, \mathbf{c}$ 均为单位向量，且 $\mathbf{a} + \mathbf{b} + \mathbf{c} = \mathbf{0}$，可知它们两两夹角均为 $120^\circ$，在空间中构成等边三角形。

**坐标法：** 取
$$\mathbf{a} = (1, 0),\quad \mathbf{b} = \left(-\frac{1}{2}, \frac{\sqrt{3}}{2}\right),\quad \mathbf{c} = \left(-\frac{1}{2}, -\frac{\sqrt{3}}{2}\right)$$

设 $\mathbf{v} = -2\mathbf{a} + t\mathbf{b} + (1-t)\mathbf{c}$，代入坐标得
$$\mathbf{v} = \left(-\frac{5}{2},\ \sqrt{3}\left(t - \frac{1}{2}\right)\right)$$

计算得
$$|\mathbf{v}|^2 = \left(-\frac{5}{2}\right)^2 + \left[\sqrt{3}\left(t - \frac{1}{2}\right)\right]^2 = 3t^2 - 3t + 7$$

</details>

<details><summary>参考解答</summary>

由题意得
$$|\mathbf{v}|^2 = 3t^2 - 3t + 7,\quad t \in [0, 1]$$

这是关于 $t$ 的二次函数，图像开口向上。

<strong>求最小值：</strong>顶点处 $t = \dfrac{1}{2} \in [0,1]$
$$|\mathbf{v}|_{\min}^2 = 3\left(\dfrac{1}{2}\right)^2 - 3\left(\dfrac{1}{2}\right) + 7 = \dfrac{25}{4}$$
$$|\mathbf{v}|_{\min} = \dfrac{5}{2}$$

<strong>求最大值：</strong>代入端点 $t = 0$ 和 $t = 1$
$$t = 0:\quad |\mathbf{v}|^2 = 7 \Rightarrow |\mathbf{v}| = \sqrt{7}$$
$$t = 1:\quad |\mathbf{v}|^2 = 3 - 3 + 7 = 7 \Rightarrow |\mathbf{v}| = \sqrt{7}$$

因此取值范围为
$$\boxed{\dfrac{5}{2} \le |\mathbf{v}| \le \sqrt{7}}$$

</details>

<details><summary>点评</summary>

本题是 2020 年浙江高中数学竞赛预赛试题，考查平面向量的坐标运算与二次函数最值问题。

<strong>关键点：</strong>

1. 利用 $\mathbf{a} + \mathbf{b} + \mathbf{c} = \mathbf{0}$ 识别出三向量两两夹角为 $120^\circ$ 的几何背景
2. 通过坐标法将向量模长问题转化为求二次函数在闭区间的最值
3. 注意二次函数顶点是否在区间 $[0,1]$ 内——本题顶点 $t = \dfrac{1}{2}$ 恰好在区间内，所以最小值在顶点取得，最大值在端点取得

</details>
