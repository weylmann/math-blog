在锐角 △ABC 中，角 A、B、C 的对边分别为 a、b、c，S 为三角形面积，满足

$$a^2 = 2S + (b-c)^2$$

求下列表达式的取值范围：

$$\frac{2\sin^2B + \sin^2C}{\sin B \sin C}$$

<details><summary>参考解答</summary>

<strong>析</strong> 本题考查解三角形中的最值问题。首先利用已知条件结合余弦定理和面积公式求出角 A 的正弦值和余弦值，然后利用正弦定理将目标表达式转化为关于边长比值的函数，最后根据锐角三角形的限制条件确定自变量范围，利用基本不等式求最值。

<strong>解</strong> 由已知条件：

$$a^2 = 2S + (b-c)^2 = 2S + b^2 + c^2 - 2bc$$

由余弦定理 $a^2 = b^2 + c^2 - 2bc\cos A$，三角形面积 $S = \frac{1}{2}bc\sin A$，代入上式得：

$$b^2 + c^2 - 2bc\cos A = bc\sin A + b^2 + c^2 - 2bc$$

整理得：

$$2bc - 2bc\cos A = bc\sin A$$

两边同时除以 $bc$（$bc \neq 0$）：

$$2 - 2\cos A = \sin A$$

即：

$$\sin A + 2\cos A = 2$$

联立 $\sin^2 A + \cos^2 A = 1$，设 $\cos A = x$，则 $\sin A = 2 - 2x$，代入得：

$$(2-2x)^2 + x^2 = 1$$

$$4 - 8x + 4x^2 + x^2 = 1$$

$$5x^2 - 8x + 3 = 0$$

解得 $x = \frac{3}{5}$ 或 $x = 1$（舍去，因为 $A$ 为三角形内角，$\cos A \neq 1$）

故：

$$\cos A = \frac{3}{5}, \quad \sin A = \frac{4}{5}$$

由正弦定理 $\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C}$，目标表达式可化为：

$$\frac{2\sin^2B + \sin^2C}{\sin B \sin C} = \frac{2b^2 + c^2}{bc} = \frac{2b}{c} + \frac{c}{b}$$

令 $t = \frac{b}{c}$，则目标函数为：

$$y = 2t + \frac{1}{t}$$

下面确定 $t$ 的取值范围。由 $t = \frac{b}{c} = \frac{\sin B}{\sin C}$，且 $B = \pi - A - C$，得：

$$t = \frac{\sin(A+C)}{\sin C} = \frac{\sin A \cos C + \cos A \sin C}{\sin C} = \sin A \cot C + \cos A$$

代入 $\sin A = \frac{4}{5}$，$\cos A = \frac{3}{5}$：

$$t = \frac{4}{5}\cot C + \frac{3}{5}$$

在锐角三角形中，$0 < C < \frac{\pi}{2}$，且 $B < \frac{\pi}{2}$，即：

$$\pi - A - C < \frac{\pi}{2} \Rightarrow C > \frac{\pi}{2} - A$$

由 $\cos A = \frac{3}{5}$，得 $\tan A = \frac{4}{3}$，故：

$$\tan C > \tan\left(\frac{\pi}{2} - A\right) = \cot A = \frac{3}{4}$$

因此 $0 < \cot C < \frac{4}{3}$，代入 $t$ 的表达式：

$$\frac{3}{5} < t < \frac{4}{5} \cdot \frac{4}{3} + \frac{3}{5} = \frac{16}{15} + \frac{9}{15} = \frac{25}{15} = \frac{5}{3}$$

即 $t \in \left(\frac{3}{5}, \frac{5}{3}\right)$。

对于函数 $y = 2t + \frac{1}{t}$（$t > 0$），由基本不等式：

$$y = 2t + \frac{1}{t} \geq 2\sqrt{2t \cdot \frac{1}{t}} = 2\sqrt{2}$$

当且仅当 $2t = \frac{1}{t}$，即 $t = \frac{\sqrt{2}}{2}$ 时取等号。

由于 $\frac{3}{5} = 0.6 < \frac{\sqrt{2}}{2} \approx 0.707 < \frac{5}{3} \approx 1.667$，故最小值 $2\sqrt{2}$ 可以取到。

再计算端点值：

当 $t \to \frac{3}{5}$ 时，$y \to 2 \cdot \frac{3}{5} + \frac{5}{3} = \frac{6}{5} + \frac{5}{3} = \frac{18+25}{15} = \frac{43}{15}$

当 $t \to \frac{5}{3}$ 时，$y \to 2 \cdot \frac{5}{3} + \frac{3}{5} = \frac{10}{3} + \frac{3}{5} = \frac{50+9}{15} = \frac{59}{15}$

由于 $t = \frac{3}{5}$ 和 $t = \frac{5}{3}$ 对应的是直角三角形边界（不满足锐角三角形条件），故端点值不可取。

综上所述，所求表达式的取值范围是：

$$\left[2\sqrt{2}, \frac{59}{15}\right)$$

<strong>【点评】</strong> 本题综合考查了余弦定理、面积公式、正弦定理、基本不等式等多个知识点。解题关键在于：①通过已知条件求出角 A 的三角函数值；②利用正弦定理将三角函数表达式转化为边长比值；③根据锐角三角形的限制条件确定自变量范围。需要注意的是，锐角三角形的条件意味着三个角都要小于 90°，这会给边长比值带来限制，端点值往往对应直角三角形边界，不能取到。

<strong>答案：</strong> C. $\left[2\sqrt{2}, \frac{59}{15}\right)$
