已知 $\dfrac{1 + \cos A}{\sin A} = \dfrac{1 + \cos B}{\sin B} + 1,\quad a = 1$，求 $\triangle ABC$ 周长的最大值。

<details><summary>参考解答</summary>

<strong>析</strong> 由条件化简得 $\sin B + \sin B\cos A = \sin A + \sin A\cos B + \sin A\sin B$，利用正弦定理将边转化为三角函数。

<strong>解</strong> 由条件得：
$$\sin B + \sin B\cos A = \sin A + \sin A\cos B + \sin A\sin B$$
利用正弦定理 $b=2R\sin B,\; c=2R\sin C$，注意到 $a=1$，则 $R=\dfrac{a}{2\sin A}=\dfrac{1}{2\sin A}$。
经化简可得周长：
$$C = a+b+c = 1 + 2\cos B + \sin B \le 1 + \sqrt{5}$$
等号在 $\sin B = \dfrac{2}{\sqrt{5}},\; \cos B = \dfrac{1}{\sqrt{5}}$ 时取到。

<strong>【点评】</strong> 本题关键在于利用正弦定理将边长表示为三角函数，再利用辅助角公式求最值。
