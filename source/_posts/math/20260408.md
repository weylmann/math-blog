## 📝 题目

> 在锐角三角形 $ABC$ 中，已知：
> $$2 \sin^2 A + \sin^2 B = 2 \sin^2 C$$
> 求 $\dfrac{1}{\tan A} + \dfrac{1}{\tan B} + \dfrac{1}{\tan C}$ 的最小值。

---

## 💡 提示

- 利用正弦定理将边长关系转化为正弦关系
- 三角形中 $\tan A + \tan B + \tan C = \tan A \cdot \tan B \cdot \tan C$
- 基本不等式（AM-GM）

---

## 📊 解析

<details>
<summary>点击查看解析</summary>

### 第一步：正弦定理转化
由 $2\sin^2 A + \sin^2 B = 2\sin^2 C$ 得：
$$2a^2 + b^2 = 2c^2$$

### 第二步：余弦定理建立角的关系
由 $c^2 = a^2 + \dfrac{b^2}{2}$，代入余弦定理：
$$\cos C = \frac{a^2 + b^2 - c^2}{2ab} = \frac{b}{4a}$$

利用正弦定理 $\dfrac{b}{a} = \dfrac{\sin B}{\sin A}$，得：
$$\cos C = \frac{\sin B}{4\sin A}$$

### 第三步：三角变换求 $\tan C$
由 $\sin B = \sin(A+C) = \sin A \cos C + \cos A \sin C$ 代入并整理：
$$\frac{3}{4}\cos C = \frac{\sin C}{4\tan A}$$
$$\Rightarrow \tan C = 3\tan A$$

### 第四步：三角形正切恒等式
在三角形中 $\tan A + \tan B + \tan C = \tan A \cdot \tan B \cdot \tan C$，代入 $\tan C = 3\tan A$：
$$4\tan A + \tan B = 3\tan^2 A \cdot \tan B$$
$$\Rightarrow \tan B = \frac{4\tan A}{3\tan^2 A - 1}$$

### 第五步：代入目标表达式
$$\frac{1}{\tan A} + \frac{1}{\tan B} + \frac{1}{\tan C}$$
$$= \frac{1}{\tan A} + \frac{3\tan^2 A - 1}{4\tan A} + \frac{1}{3\tan A}$$
$$= \frac{13}{12\tan A} + \frac{3}{4}\tan A$$

### 第六步：AM-GM 不等式
$$\frac{13}{12\tan A} + \frac{3}{4}\tan A \geq 2\sqrt{\frac{13}{12\tan A} \cdot \frac{3}{4}\tan A} = \frac{\sqrt{13}}{2}$$

当且仅当 $\dfrac{13}{12\tan A} = \dfrac{3}{4}\tan A$，即 $\tan^2 A = \dfrac{13}{9}$ 时取等。

</details>

**解**：见解析

**答案：** $\displaystyle \frac{\sqrt{13}}{2}$

## 📚 解题总结

### 思路分析
1. **正弦定理**将边长关系转化为正弦关系
2. **余弦定理**建立角之间的比例关系
3. **三角恒等变换**求出 $\tan C = 3\tan A$
4. 利用**正切恒等式**消元
5. **基本不等式**求最值

### 方法总结
- 关键技巧：边角互化、正切恒等式 $\tan A + \tan B + \tan C = \tan A \tan B \tan C$
- 核心思想：消元降维，统一变量

### 启示
处理三角形中涉及正切的多元最值问题，常用正切恒等式进行消元。
