第一步：安装 OpenClaw
最新版本内置 QQ 机器人支持

🖥️ Windows 安装

npm install -g openclaw@latest

需要 Node.js 18+ 环境

✅ 验证安装

openclaw --version

查看当前版本号

📋 前置要求

  • Node.js 18 或更高版本
  • npm（通常随 Node.js 一起安装）
  • 稳定的网络连接
5 / 13
第二步：配置 OpenClaw
运行配置向导，设置 AI 模型（推荐 MiniMax）

⚙️ 运行配置向导

openclaw configure

🔧 配置内容

  • 选择语言（中文）
  • 选择 AI 模型（推荐 MiniMax）
  • 配置 QQ 机器人（下一步详细说明）
  • 配置消息通道

🤖 支持的模型

  • MiniMax（推荐）
  • 阿里通义千问
  • 百度千帆
  • 硅基流动
6 / 13
第三步：配置 QQ 机器人
新版 OpenClaw 已内置 QQBot 通道

📝 配置文件示例

channels: {
  qqbot: {
    enabled: true,
    appId: "你的AppID",
    appSecret: "你的AppSecret",
    token: "你的Token"
  }
}

🔑 获取 QQ 机器人凭证

  • 前往 QQ 开放平台（q.qq.com）
  • 创建机器人应用
  • 获取 AppID、AppSecret
  • 配置机器人权限

✅ 验证配置

openclaw gateway restart

重启后查看日志确认 QQBot 是否加载成功

7 / 13
Obsidian：题目收集
语音输入 → 自动转文字 → AI 整理入库

📱 收集方式

  • 语音消息 → 直接说"记录这道题"
  • 截图 → AI OCR 识别题目
  • 文字复制 → AI 自动格式整理
  • 手写拍照 → AI 识别公式

🗂️ 自动整理

  • 按知识点分类（函数/向量/几何…）
  • 打标签（难度、年级、章节）
  • 自动关联已有题目
  • 记录题目来源
8 / 13
Obsidian：任务提醒
所有待办集中管理，再也不会漏
  • 说"记录一下" → 自动创建待办事项，带时间戳
  • 日记中记录的任务 → 自动提取为待办清单
  • 到期提醒 → 推送通知到 QQ
  • 任务状态追踪 → 完成/未完成一目了然
"口头交代 → 语音转文字 → AI 整理 → 准时提醒"

适合哪些老师？
  • 需要 持续输出内容（每日一题、周测、月考）的老师
  • 有大量学生答疑记录需要整理的老师
  • 想要系统化管理教学资料的老师
  • 希望减少重复劳动、专注教学本身的老师

💡 核心价值

不是用 AI 替代老师，而是用 AI 处理信息管理，
让老师把时间和精力放在 教学设计 和 学生互动 上。
